# Day 3:- 08/07/2026 | Time Spent: 4 Hours
Journal:- Today I started working on my Coders Peace Lamp Project. I collected all parts at one place and started working on it. 
first I rmoved Upcycled Earbud Charging Dock PCB from earbud box. Then I take one small charging cable which I cut from one side which we can see in Work Image Folder. then that cut cable connected to Direct 5V Blue-Spectrum LED Capsule. But that light was too small and for some minutes I stopped working on it but then I got one more Idea that what if I coonect one more light but that will not capsule that lite will decoration lighting series type, I once again check my storage box where I store some useless or waste things and there I found the light which was already connector of that Earbud charging dock but on that days that was broken so I throw and stored it in my storage box but for to connect that lite to dock I need Upcycled Micro Ribbon-Connector and I go to my friend's because last time I saw that he has that connecter and I take Micro Ribbon-Connector from him and assemble it with lite and dock. Now Project is completed but not tested. In that dock there it has two port one Input and second output and I connected capsule lights cable to output and then i connected Tethered USB Power Cable to Input port and Tethered USB Power Cable connected to Laptop. 
Testing is ready now, Project is working 100%.
