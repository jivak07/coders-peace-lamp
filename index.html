Final Design Specification

  • Project Name: Coders Peace Lamp (Final Project Plan) Form Factor: Stand-independent, 360-degree adjustable flexible iron gooseneck chassis with integrated USB-A power input. Operates seamlessly on host laptop power, power banks, or standard 5V wall adapters. 1. Dual-Zone Advanced Lighting Architecture:- Zone 1: Front Focus Spotlight: Equipped with a directional optical lens/collimator that concentrates a tight beam of light forwards (similar to a tactical beam but using multiple integrated color wavelengths). Zone 2: Side Ambient Light Row: A linear diffusion panel running along the side of the cylindrical lamp head. It houses a multi-color LED matrix that casts a wide-angle, soft ambient glow across the desk workspace. 2. Multi-State Sequential Control Logic (Dual UI Buttons):- The system eliminates swappable lenses by embedding all color hardware internally. Two independent tactile micro-switches control the state machine of the microcontroller:- Button 1: Manages Front Focus Spotlight Button 2: Manages Side Ambient Light Row Sequential Click Cycling Logic (For both buttons independently):- 1st Click: Blue Light (High-frequency focus mode / Melatonin suppression) 2nd Click: Green Light 3rd Click: Red Light 4th Click: Yellow Light 5th Click: White Light 6th Click: All LEDs Active Simultaneously (Maximum Lumens Output Mode) 7th Click: System Power Off / Reset State

Phase 2: The Future Upgrade

Currently, this is an MVP made from 100% e-waste. But my vision for Phase 2 is to upgrade this into a Professional Room Projection Lamp.


  • RGB Crystal Projection: Upgrading the light node to a lens projector (like commercial room decor lamps) to spread focus-enhancing spectrums across the ceiling.
  • Psychology & Mode Shifting: Using a Raspberry Pi Pico to create different light profiles. A "Deep Work" mode for coding, and a relaxing mode for winding down.
  • Total Portability: Adding a 18650 Li-ion battery so coders can work anywhere, even during long power cuts in rural areas.
